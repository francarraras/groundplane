# Operating Context Intake

Use this as a local scratchpad if chat is inconvenient. Paste rough, messy notes between the markers. Command Center will turn them into reviewable proposals before anything is promoted to durable state or memory.

Recommended first scope: **Active Projects**.

## Active Projects

Useful fields:

- project or commitment name
- why it matters
- current status
- next action if known
- deadline or urgency if any
- blocker or worry if any

## Daily Routine

Useful fields:

- wake/work/sleep anchors
- recurring responsibilities
- what usually derails the day
- what the assistant should enforce
- hard constraints

## Life Areas

Useful fields:

- main life areas
- what each area is responsible for
- current pain or risk
- desired standard
- anything private or gated
